===================================================================
ОПИСАНИЕ ПРОЕКТА И СТРУКТУРА
===================================================================
Консольная программа на языке C для вывода содержимого файлов 
в шестнадцатеричном виде под управлением ОС Windows.

Особенности реализации:
* Осуществляет преобразование байт файла в hex-символы вручную (без %x/%X).
* Поддерживает смещение (-o), лимит чтения (-l), группировку (-g) и ширину строки (-n).
* Автоматически разворачивает байты внутри групп по принципу Little Endian.
* Дополняет неполные группы нулями "00" в конце файла.
* Выводит красивую ASCII-колонку символов справа для размера групп g = 1.
* Обходит и обрабатывает целые директории (-d) через системный Windows API.

Структура проекта:
* app/ — исполняемое приложение (разбор CLI-аргументов через getopt и валидация).
* lib/ — библиотека функций (чтение файлов, hex-преобразователь и обход папок).

===================================================================
ПОДРОБНАЯ ИНСТРУКЦИЯ ПО СБОРКЕ И ЗАПУСКУ В POWERSHELL (WINDOWS)
===================================================================

1. Очистка кэша и сборка проекта

# Удаляем старую папку сборки (если она была), чтобы избежать конфликтов кэша:
> Remove-Item -Recurse -Force build

# Создаем чистую папку для сборки и переходим в нее:
> mkdir build
> cd build

# Настраиваем CMake:
> cmake ..

# Запускаем компиляцию проекта:
> cmake --build . --config Debug


2. Ручной запуск программы и тестирование опций

После успешной сборки исполняемый файл "my_program.exe" появится в подпапке 
"build/app/Debug/". Все команды ниже выполняются из папки "build".

* Пример 1. Вывод первых 20 байт файла "CMakeLists.txt" (g = 1, превью справа):
  > ./app/Debug/my_program.exe -i ../CMakeLists.txt -l 20

* Пример 2. Вывод с пропуском первых 10 байт (смещение) и лимитом в 16 байт:
  > ./app/Debug/my_program.exe -i ../CMakeLists.txt -o 10 -l 16

* Пример 3. Группировка байт по 4 штуки (Little Endian) и вывод по 2 группы в строке:
  > ./app/Debug/my_program.exe -i ../CMakeLists.txt -l 11 -g 4 -n 2

* Пример 4. Вывод всех файлов в папке "app/src" (лимит по 15 байт на каждый файл):
  > ./app/Debug/my_program.exe -d ../app/src -l 15

* Пример 5. Вызов встроенной справки по доступным флагам:
  > ./app/Debug/my_program.exe -h
